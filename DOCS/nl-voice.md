# Hooklyne NL voice guide

**Owner:** Joey + Tim
**Last updated:** 26 april 2026

This is the reference for writing native Dutch on Hooklyne. Read it before touching any `nl:` string. The goal: copy that reads as if it was written in Dutch in the first place, not translated from English.

---

## Sentence structure

- **Short.** Period-heavy. Fragments OK.
- Drop subordinate clauses where you can. Two short sentences beat one long one.
- Active voice. "We doen X" not "X wordt door ons gedaan".
- Lists work. Tables work. Three-word headers work.

## Word choice

- **"Je" not "u"** unless writing to enterprise legal.
- **"Jouw" only emphatic** ("DAT is JOUW probleem"), normal possession is "je".
- Anglo-Saxon-rooted Dutch over Latinate where natural:
  - "doen" > "uitvoeren"
  - "maken" > "creëren"
  - "schrijven" > "redigeren"
  - "snel" > "rapide"
  - "klaar" > "voltooid"
- **Loanwords kept where natural in Dutch B2B SaaS:** sales, outreach, signal, prospect, pipeline, lead, ICP, AVG, MKB, B2B, credits, seats, deliverability, hook, sender profile, RFP, NPD, CRM, sequencer, hire, outbound, tooling.

## Constructions to use

- **"Voor [doelgroep] die [gedrag]"** - "Voor sales reps die slimmer willen werken"
- **"Eén [iets] voor [gebruik]"** - "Eén plek voor je hele pipeline"
- **"Geen [iets]. Geen [iets]."** - "Geen ticket. Geen wachtrij. Geen chatbot."
- **Imperatives** - "Probeer het", "Bekijk het", "Boek een call"
- **Direct second person** - always "je [doet iets]"
- **Numbers as facts** - "20+ providers", "Onder de minuut"

## Calques to avoid (these read as translated)

| ❌ Calque | ✅ Native |
|---|---|
| "tot we groeien, krijg je ons" | "klein zijn heeft zo zijn voordelen" |
| "leest alsof je rep het schreef" | "lees het. Het klinkt als je rep" |
| "het meeste van die jaren ging op aan" | "de meeste tijd ging op aan" |
| "Hooklyne is wat je krijgt als je stopt met..." | "Hooklyne is daar het antwoord op" |
| "we zaten zelf op die stoel" | "we deden tien jaar wat jij nu doet" |
| "in een vacuüm bedacht" | (drop entirely, just state the thing) |
| "elke beweging gescoord tegen" | "we scoren elke beweging tegen" |
| "wordt gevormd door jou" | "jij stuurt 'm" |
| "wordt afgestemd per regio" | "we stemmen 'm af per regio" |
| "een lijst van elke limiet" | "een lijst met wat misging" |
| "jouw voor ons telt zwaarder dan" | "jouw [iets] telt voor ons zwaarder dan" |

- "Onthul"
- "Maximaliseer" / "Optimaliseer"
- "Ontdek de magie"
- "Krachtige"
- "Baanbrekend"
- "Revolutionair"
- "In een handomdraai"
- "Haal het maximale uit"
- "Verbeter je [iets]"
- "Beleef de [iets]"

- "State-of-the-art"
- "Cutting-edge"

## Capitalization

- **Months and weekdays: lowercase.** "februari 2026", not "Februari 2026"
- **Languages, nationalities: capitalized.** "Nederlands", "Engels", "Brits"
- **Loanword job titles: keep their casing.** "VP Ops", "Head of Procurement"
- **Sentence-case headlines.** Only first word + proper nouns capitalised
- **Brand names: as-is.** "Hooklyne", "Apollo", "Cognism", "Lusha", "ChatGPT"

## Formatting

- **Comma as decimal separator.** "€0,85" not "€0.85", "3,2 MW" not "3.2 MW"
- **Period as thousands.** "80.000 huishoudens"
- **"mnd" in tight slots, "maand" in flowing prose**
- **No em dashes.** Use ` - ` or rewrite the sentence.

## When in doubt

Ask: "Would Tim say this in Dutch on a sales call?" If yes, keep it. If you'd write it differently in your head and translate it, rewrite it.
